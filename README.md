# 自作SFA
SFA（Sales Force Automation）と書いていますが、なんとなく雰囲気をまねたものです。

## OverView
非IT系なチームがExcelで管理してそうな、日々蓄積される情報をまとめたいと思い作ってみました。

## Demo
https://imitate-sfa.herokuapp.com/

## Description
各項目に対してデータを表示、追加、編集、削除できます。

## Features
天気の項目は、気象庁からデータを自動で取得しています。

## Requirement
* Python 3
* Django 3
* Bootstrap 4
